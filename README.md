# Автотесты для сервиса «Яндекс Самокат»

Учебный проект 4 спринта курса по автоматизации тестирования на Java.  
Проект содержит UI‑автотесты для учебного веб‑приложения «Яндекс Самокат»:
https://qa-scooter.praktikum-services.ru/

## Технологии

- Java 11
- Maven
- JUnit 4
- Selenium WebDriver
- WebDriverManager

## Что покрыто тестами

### Раздел «Вопросы о важном» (FAQ)

- Проверка, что при клике на каждый из 8 вопросов открывается ответ.
- Проверка, что текст ответа совпадает с ожидаемым.

Тесты находятся в пакете: `ru.yandex.practicum.scooter.faq`.

### Позитивный сценарий заказа самоката

- Две точки входа в форму заказа:
  - верхняя кнопка «Заказать»;
  - нижняя кнопка «Заказать».
- Два набора тестовых данных (имя, фамилия, адрес, метро, телефон, дата, срок аренды, цвет самоката, комментарий).
- Проверка:
  - корректного заполнения полей формы;
  - выбора цвета самоката;
  - появления модального окна с подтверждением заказа.

Тесты находятся в пакете: `ru.yandex.practicum.scooter.order`.

## Структура проекта

- `src/main/java/ru/yandex/practicum/scooter/pageobject` — Page Object классы:
  - `HomePage` — главная страница;
  - `OrderPageStepOne` — первая часть формы заказа;
  - `OrderPageStepTwo` — вторая часть формы заказа.
- `src/test/java/ru/yandex/practicum/scooter` — базовый класс тестов `TestBase`.
- `src/test/java/ru/yandex/practicum/scooter/faq` — тесты FAQ.
- `src/test/java/ru/yandex/practicum/scooter/order` — тесты оформления заказа.

## Как запустить тесты

1. Установите Java 11 и Maven.
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/zarina-gonyukova/Sprint_4.git

   ```
3. Убедитесь, что установлен Google Chrome.
4. Запустите все тесты командой:
   ```bash
   mvn test
   ```

По умолчанию тесты запускаются в браузере Chrome (управление драйвером выполняет WebDriverManager).

### Запуск отдельных групп тестов

Запуск только тестов FAQ:

```bash
mvn -Dtest=FaqTest test
```

Запуск только тестов оформления заказа:

```bash
mvn -Dtest=OrderScooterTest test
```

## Известные особенности

- В учебном стенде «Яндекс Самокат» есть баг, который иногда не позволяет успешно оформить заказ в браузере Chrome.  
  В этом случае тест оформления заказа может падать на шаге подтверждения, несмотря на корректную работу автотеста.   
- Тесты используют неявное ожидание и явные ожидания для модального окна заказа; при медленном интернете время ожидания может оказаться недостаточным и потребовать увеличения таймаутов.
